
Cara Guna Sniper Bot Maria:

1. Upload semua file ni ke repo GitHub Boss (contoh: sniper-bot-maria)
2. Pergi ke Railway → New Project → Deploy from GitHub
3. Pilih repo yang Boss upload tadi
4. Masukkan 2 environment variable:
   - TELEGRAM_TOKEN = [Token Bot]
   - TELEGRAM_CHAT_ID = [Chat ID]
5. Railway akan run & bagi webhook URL
6. Masukkan webhook tu dalam TradingView alert

Siap!
